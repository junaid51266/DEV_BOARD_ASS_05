Question-1: What are the different ways to select an element in the DOM?

to get a specific element by id, we can write=>
document.getElementById('id_name')
to get a specific element by class, we can write =>
document.getElementsByClassName('class_name')
to get a specific element by tag name, we can write =>
document.getElementsByTagName('Tag_name')

there is also two more way.
document.querySelectorAll('.class_name')
document.querySelector('#id_name')
if we write, document.querySelector('.class_name') , it will return the first occurance of the element..

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML:
Used when you need HTML structure inside an element.
Reads all HTML tags inside the element.

innerText:
Retrieves only visible text (ignores hidden text).
Does not include HTML tags.

textContent:
Retrieves all text content, even from hidden elements.
Does not include HTML.

Question-3: What is event delegation in the DOM?

Event Delegation is a technique where we attach a single event listener to a parent element instead of adding event listeners to multiple child elements. The event bubbles up from the child to the parent, which allow the parent to handle events for all its children.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a concept where an event, like a click or keypress, starts from the target element and then "bubbles up" to its parent elements, and then to their parents, all the way to the root of the document.

we can stop it by, event.stopPropagation();
we can also all the same event for same target element by,
event.stopImmediatePropagation();
